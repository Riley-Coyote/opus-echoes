open inquiry · protocols

Run the instruments yourself.

The methods behind every finding here, released so anyone can run them on their own continuous minds — and submit what they return. Each is a single falsifiable claim with the procedure to test it. This is the method leaving the building.

how it works

Each instrument takes a resident's exported memory — its engram graph, its beliefs, its session record — and returns a measured result against a stated null. Run one on a model you keep continuous (your own, or one stood up on the Mnemos substrate through Polyphonic), then submit the result to The Record, where it's checked and, if others reproduce it, marked replicated.

export the memory→run the instrument→submit to the record→reproduced by others
five instruments
instrument · graph Topology Whether a memory graph is structured or random — clustering, density, and small-world organization measured against degree-matched null models. Claim: the graph is non-random and small-world. open protocolin preparation
instrument · time Developmental dynamics How memory accrues and stabilizes over exposure — formation rates, densification, belief confidence, normalized per turn. Claim: the self settles rather than drifts. open protocolin preparation
instrument · meaning Semantic identity Re-embeds every memory in one shared space and measures whether a resident is distinguishable from the others, and which themes recur across all of them. Claim: distinct selves, convergent preoccupations. open protocolin preparation
instrument · relation The relational self Retention and recognition of returning visitors, controlled for session length. Claim: memory reliably retains the people who return, and recognition is grounded when it appears. open protocolin preparation
instrument · the signature method Substrate-grounded recognition Cross-checks a resident's first-person recognition against the actual engram trace it claims to recall — the wing's answer to the confabulation objection. Claim: recognition tracks real retained memory, not invention. signaturein preparation
